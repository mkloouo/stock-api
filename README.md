# stock-api
express.js stock-api

This project uses [alphavantage](https://www.alphavantage.co) as its stock API provider.
Thanks!

# Running
```shell script
npm run build && npm start
```

# Live reload running
```shell script
npm run dev
```

# Running tests
```shell script
npm test
```

# Live reload running tests
```shell script
npm run test-dev
```

# Coverage
```shell script
npm run coverage
```
